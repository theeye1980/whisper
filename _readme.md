Описание скриптов:

wisper_OpenAI.py - расшифровывает файлы из указанной папки. Размещает по каждому файлу из папки отдельную расшифровку. Затем собирает все текстовые файлы, сортирует по алфавиту и склеивает в один выходной файл, одноименный с папкой. 

spliter_mass.py - указываем папку с mp3 файлами. Он создаст папку в корне по каждому файлу, а сам файл поделит на части по указанному на входе временному отрезку. По умолчанию стоит 600 секунд.

spliter.py - на входе mp3 файл. На выходе папка. Скрипт файл поделит на части по указанному на входе временному отрезку. По умолчанию стоит 600 секунд.

reassemble.py - пересобирает в выходной общий файл все текстовые файлы из папки. Нужен для запуска после коррекции некоторых частей
